# Configuration

## Apple
Environment variables that can be configured:

```dotenv
# Fastlane username and password
# https://docs.fastlane.tools/best-practices/continuous-integration/#environment-variables-to-set
FASTLANE_USERNAME=
FASTLANE_PASSWORD=

# Apple
FL_APP_IDENTIFIER=
FL_TEAM_ID=
FL_ITC_TEAM_ID=
```
